UtilityBenefits.com — clickable prototype

Design system + electricity and internet verticals fully wired (pillar → assistance → key cluster pages), interactive eligibility calculator, stack recommendations.

Built against Master Spec v1.7. v1.1 adds the internet vertical including the strategic "What replaced ACP?" page.

How to use

Click any tile below to open that page. Some pages link to others — full nav works between them. The recommended walkthrough order is numbered.

  • Recommended walkthrough: 1 (homepage) → 2 (electricity pillar) → 8 ("What replaced ACP?" — the strategic page) → 10 (calculator) → 11 (stack doc)
  • Calculator URL params work: e.g. qualify/index.html?state=tx pre-fills Texas
  • Crisis path in calculator: pick "I have a shutoff notice" at step 5 to see path D
  • Try the household size + categorical eligibility shortcut: any of SNAP/SSI/TANF/Medicaid skips the income step
